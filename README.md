# tdarr-exporter

## Background
`tdarr-exporter` is a Prometheus collector for [Tdarr](https://github.com/HaveAGitGat/Tdarr) and provides the following as Prometheus metrics

1. General instance statistics (as shown in Tdarr's statistics page).
2. Filterable metrics by instance, library, and node.
3. Current queues and counts.
4. General resource usage of tdarr nodes.
5. Progress updates on running transcode and health check jobs.

Inspired by exportarr and qbittorrent-exporter projects.
